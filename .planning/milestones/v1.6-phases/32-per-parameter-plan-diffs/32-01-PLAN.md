<objective>
Enrich `rig plan` output with per-parameter diffs so the user can see exactly which knob
positions or CC values will change — not just which scene or device changed.

Purpose: The current output tells users "this device changes" but not what changes. A guitarist
about to apply a scene needs to know which specific parameters differ so they can verify the
result. Parameter-level visibility also makes `rig plan` useful for reviewing accumulated config
drift before applying.

Output:
- `ParamDiff` Pydantic model added to `models.py`
- `param_diff: list[ParamDiff]` field on `DeviceAction` (defaults to `[]`)
- Two private helpers in `compute.py`: `_find_preset` and `_compute_param_diff`
- `compute_plan` wired to populate `param_diff` for ANALOG and CONFIGURE actions; VERIFY actions
  always receive `[]`
- CLI `plan.py` renders param diff sub-lines under ANALOG and CONFIGURE action lines
- JSON output gains `param_diff` automatically via Pydantic serialization
- `TestParamDiff` class in `test_plan.py` with five scenarios
- CLI tests in `test_cli_plan.py` validating text and JSON param diff rendering
</objective>

    Add both helpers to compute.py after the existing _get_preset_number function. Import

    Implement _find_preset:

        def _find_preset(rig: Rig, pedal_id: str, preset_id: str | None) -> object | None:
            if preset_id is None:
                return None
            device = rig.devices.get(pedal_id)
            if device is None:
                return None
            return next((p for p in device.presets if p.id == preset_id), None)

    Implement _compute_param_diff:

        def _compute_param_diff(before_preset: object | None, after_preset: object) -> list[ParamDiff]:
            after_params: dict[str, float | str | bool] = {}
            if hasattr(after_preset, "values"):
                after_params = after_preset.values
            elif hasattr(after_preset, "parameters"):
                after_params = after_preset.parameters
            if not after_params:
                return []
            before_params: dict[str, float | str | bool] = {}
            if before_preset is not None:
                if hasattr(before_preset, "values"):
                    before_params = before_preset.values
                elif hasattr(before_preset, "parameters"):
                    before_params = before_preset.parameters
            diffs: list[ParamDiff] = []
            for key, after_val in after_params.items():
                before_val = before_params.get(key)
                if before_preset is None or before_val != after_val:
                    diffs.append(ParamDiff(name=key, before=before_val, after=after_val))
            return diffs
